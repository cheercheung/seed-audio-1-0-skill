# Security Policy

## Supported Scope

This repository contains documentation, examples, an npm installer, and local scripts for the EvoLink Seed-Audio API.

## Reporting a Vulnerability

Do not open a public issue for secrets, API keys, credential exposure, or vulnerabilities that could affect users.

Report security concerns through the EvoLink maintainer contact route. Include:

- affected file or command
- impact
- reproduction steps
- proposed fix, if available

## API Keys

Never commit `EVOLINK_API_KEY`, `.env`, shell history, generated credentials, or private callback URLs.
